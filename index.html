Sobre o Projeto

Este sistema recebe valores de vendas, calcula automaticamente a comissão devida e gera planilhas processadas. Desenvolvido em Balneário Camboriú, focado em segurança e integridade de dados.

☁️ Hospedagem com PythonAnywhere

O backend da aplicação, incluindo os scripts de cálculo e verificação, será executado na plataforma PythonAnywhere. Isso garante um ambiente de servidor estável e seguro para processar os dados.

🛠️ Componentes do Sistema

Arquivo Função Principal

app_planilha_sign_data_.pyCarimbo Digital (Assinatura dos dados).

🚀 Fluxo de Trabalho (Commit)

1. Atualizar

Insira novos dados no CSV e salve as alterações.

2. Assinar

Rode o script de assinatura para travar os novos dados.

3. Validar

Execute o verificador para garantir sucesso antes do Push.

🔐 Segurança Crítica

A Chave Privada está protegida fora deste repositório para evitar vazamentos acidentais. O sistema utiliza o padrão RSA-PSS para máxima proteção.
